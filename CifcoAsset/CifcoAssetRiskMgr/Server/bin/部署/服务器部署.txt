服务器部署步骤：
一、数据库准备：
1.安装oracle数据库（10g或以上版本），安装时选择中文语言包。
2.在SQLPlus中，以system用户连接到oracle，并执行"Create User.sql"脚本，为风控系统创建表空间和用户。
3.导入risk1.dmp文件。打开命令窗口，执行如下命令："imp riskuser/riskpwd@sid file="d:\maaccount.dump" fromuser=risk1 touser=riskuser"，其中，file为存放dump文件的路径，sid为步骤1中安装的oracle数据库的sid，riskuser为步骤2中脚本文件创建的风控用户名，riskpwd为风控用户密码。

二、运行环境准备：
1.在准备运行服务程序的机器上安装oracle 11g客户端，版本号为11.2。安装时选择安装类型为管理员。
2.配置连接字符串。在oracle 11g安装目录中找到tnsnames.ora文件（通常在product\11.2.0\client_1\network\admin目录下），添加如下内容：
ORCL =
  (DESCRIPTION =
    (ADDRESS_LIST =
      (ADDRESS = (PROTOCOL = TCP)(HOST = 步骤一中oracle机器IP)(PORT = 1521))
    )
    (LOAD_BALANCE = YES)
    (CONNECT_DATA =
      (SERVER = DEDICATED)
      (SERVICE_NAME = 步骤一中oracle数据库sid)
      (FAILOVER_MODE =
        (TYPE = session)
        (METHOD = basic)
        (RETRIES = 180)
        (DELAY = 2)
      )
    )
  )

三、填写相关配置文件并安装运行服务
1.打开system.xml文件，填写好服务程序运行的端口号，数据库服务名（步骤二中的orcl）、风控用户名、风控用户密码。
