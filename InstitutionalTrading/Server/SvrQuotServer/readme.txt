行情服务器模块

功能：
	1. 连接ctp行情服务器，定制并获取所有合约的实时行情;
	2. 为服务端和客户端提供行情定制/推送服务
	3. 这是一个基于uniServer运行的dll模块
	
实现：
	1. 从DB中读取行情服务器连接参数，包括ctp行情服务器的地址、经纪公司代码、用户名、秘密啊、连接时间和断开时间
	2. 在指定连接时间后，从DB中取当日全部合约代码。如果DB中合约代码不全，等待4s再取，直到成功取到
	3. 得到合约代码后，连接ctp行情服务器。对连接状态进行超时判断，如果不在已连接状态超过60秒，则断开，重新连接
	4. 建立multimap<int,string>, 保存【sockid -> 合约名称】的映射关系，表示一个socket连接订阅了哪些合约的行情
	5. 处理tcp服务接收到的数据包。响应登录、登出、订阅、退订四个指令
	6. 在新行情到来时，推送给订阅的socket终端。(优先推送给报盘机)

接口：
	1. 向报盘机和客户端提供登录、登出、订阅、退订接口

约定：
	1. 此版本暂时不对用户登录进行验证。
	2. 保持最小依赖性，仅依赖于机构版中的SvrTcp、SvrDBOpr、DB中保存合约和ctp行情服务器账号的表
